# Atom Shortcuts

## Intro
To improve the efficiency of using Atom,
a number of useful shortcuts is noted,
referring existing works such as:
https://www.jianshu.com/p/e33f864981bb
https://www.rainyleo.com/2016/08/13/快捷键/
