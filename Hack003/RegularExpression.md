# Regular Expression
http://www.runoob.com/regexp/regexp-syntax.html

## Methods
Method | Description
---|---
re.match | match from the beginning
re.search | search the first matched substring
re.sub | replace the matched substring

## Patterns

Pattern | Description
---|---
. | any character except line break
[...] | listing characters
* | 0 or more
+ | 1 or more
\w | '[A-Za-z0-9_]'


## Examples

Example | Description
---|---
